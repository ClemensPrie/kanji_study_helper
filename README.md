# JLPT Grammar Practice App

A mobile-friendly web app to generate Japanese practice sentences using DeepSeek AI.

## Features
- **JLPT N5/N4/N3** grammar and vocabulary
- **Sentences mode**: individual sentences per grammar point
- **Short Text mode**: one coherent paragraph using all selected grammar
- **Pick specific grammar** points to practice
- **Smart vocabulary**: sends relevant level vocab to guide the AI
- **English translations** toggle
- **History** of generated content
- Works offline with all data embedded

## How to Use
1. Open https://YOUR-USERNAME.github.io/jlpt-grammar/ (after deploying)
2. Enter your DeepSeek API key in Settings
3. Select a level, optionally pick grammar points
4. Choose mode (Sentences or Short Text) and tap Generate

## Deploy to GitHub Pages
1. Create a repo named YOUR-USERNAME.github.io (or any repo name)
2. Upload all files in this folder
3. Enable GitHub Pages in repo Settings > Pages
4. Done!

## API Key
You need a DeepSeek API key. Get one at platform.deepseek.com.
API calls are very cheap: ~.0001 per generation.
